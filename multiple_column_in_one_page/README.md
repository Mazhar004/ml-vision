# Image to Text
Text extract &amp; reformatting for different text structure


## Installation
- `pip install -r requirements.txt`
### Python Version

- Python == 3.8 (Any version of Python3 will work fine)

### Library Installation

#### Linux
- `sudo apt-get install tesseract-ocr`
