# ML Vision
## Installation

### Python Version

- Python == 3.8 (Any version of Python3 will work fine)

### Library Installation
#### Linux
- Virtual Environment
  - `python3 -m venv venv`
  - `source venv/bin/activate`
- Library Install
  - `pip install --upgrade pip`
  - `pip install --upgrade setuptools`
  - **Requirements Library based on folder**
